DataLens — Agentic Data Catalog Assistant

Project brief:

A context-aware agentic assistant for an enterprise data catalog: answers natural-language questions with grounded, column-level citations and a confidence score on every answer. A deterministic governance guard hard-blocks unsafe queries before any LLM sees them, and a critic loop enforces groundedness with bounded retries (LangGraph guard → planner → executor → synthesizer → critic). Hybrid retrieval via ChromaDB + BM25 with reciprocal-rank fusion; generated SQL executes against a real store.
